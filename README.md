## Notebook Summaries

### 1. Activation Functions

File: [final_activation_functions_tutorial.ipynb](final_activation_functions_tutorial.ipynb)

This notebook explains why neural networks need non-linearity and compares the most important activation functions used in practice.

### 2. CNN Fundamentals

File: [final_cnn_fundamentals_tutorial.ipynb](final_cnn_fundamentals_tutorial.ipynb)

This notebook builds convolutional neural networks from first principles and shows how image models extract spatial features.

### 3. Hyperparameter Tuning

File: [final_hyperparameter_tuning_tutorial.ipynb](final_hyperparameter_tuning_tutorial.ipynb)

This notebook shows how tuning choices such as learning rate and batch size can dramatically change training quality.

### 4. Important Classification Metrics

File: [final_important_classification_metrics_tutorial.ipynb](final_important_classification_metrics_tutorial.ipynb)

This notebook explains how to evaluate classification models correctly instead of relying on accuracy alone.

### 5. Modern CNN Architectures

File: [final_modern_cnn_architectures_tutorial.ipynb](final_modern_cnn_architectures_tutorial.ipynb)

This notebook focuses on the architectural improvements that made deeper CNNs practical and performant.

### 6. Optimizers for Deep Learning

File: [final_optimizers_deep_learning_tutorial.ipynb](final_optimizers_deep_learning_tutorial.ipynb)

This notebook explains how optimizers guide training and why their behavior shapes convergence.
